# IWDTFSimpleIOStressAction2::Continue


## -description

Continues the I/O.

## -parameters

### -param pResult

#### - pbResult [out, retval]

True if the I/O operation succeeded; otherwise, false.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
