# WppRecorderLogDelete macro


## -description


The <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wpprecorder/nf-wpprecorder-wpprecorderlogdelete">WppRecorderLogDelete</a> method deletes the specified recorder log.


## -parameters




### -param RecorderLog

Handle to the recorder log to delete.


## -remarks



When a thread enters this function, no threads can subsequently  log to this buffer.
