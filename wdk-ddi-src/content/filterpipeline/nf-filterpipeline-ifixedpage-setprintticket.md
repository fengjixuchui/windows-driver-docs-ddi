# IFixedPage::SetPrintTicket


## -description


The <b>SetPrintTicket</b> method associates a print ticket with the page.


## -parameters




### -param ppPrintTicket






#### - pPrintTicket [in]

The print ticket object.


## -returns



<b>SetPrintTicket</b> returns an <b>HRESULT</b> value.
