# IPartFont::SetFontContent


## -description

The <b>SetFontContent</b> method sets the content of the font.

## -parameters

### -param pContentType

#### - contentType [in]

The type of the content for the font to be set.

## -returns

<b>SetFontContent</b> returns an <b>HRESULT</b> value.
