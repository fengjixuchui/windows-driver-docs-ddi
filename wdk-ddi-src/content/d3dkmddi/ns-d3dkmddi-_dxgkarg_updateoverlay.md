# _DXGKARG_UPDATEOVERLAY structure


## -description


The DXGKARG_UPDATEOVERLAY structure describes parameters for modifying an overlay.


## -struct-fields




### -field OverlayInfo

[in] A <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/d3dkmddi/ns-d3dkmddi-_dxgk_overlayinfo">DXGK_OVERLAYINFO</a> structure that contains parameters that are required to modify the overlay.
