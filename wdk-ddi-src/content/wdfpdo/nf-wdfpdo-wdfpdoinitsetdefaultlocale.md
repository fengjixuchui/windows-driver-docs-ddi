# WdfPdoInitSetDefaultLocale function


## -description

<p class="CCE_Message">[Applies to KMDF only]</p>

The <b>WdfPdoInitSetDefaultLocale</b> method sets a device's default locale.

## -parameters

### -param DeviceInit 

[in]
A pointer to a <a href="/windows-hardware/drivers/wdf/wdfdevice_init">WDFDEVICE_INIT</a> structure.

### -param LocaleId 

[in]
A locale identifier (LCID) that represents the default locale. For more information, see <a href="/windows/desktop/Intl/locale-identifiers">Locale Identifiers</a>.

## -remarks

When the system requests device description text from a driver, it specifies a locale. If text for the specified locale is not available, the framework returns device text for the driver's default locale.

The driver must call <b>WdfPdoInitSetDefaultLocale</b> before calling <a href="/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdevicecreate">WdfDeviceCreate</a>. For more information about calling <b>WdfDeviceCreate</b>, see <a href="/windows-hardware/drivers/wdf/creating-a-framework-device-object">Creating a Framework Device Object</a>.


#### Examples

The following code example sets a device's default locale to United States English (0x409).

```cpp
WdfPdoInitSetDefaultLocale(
                           pDeviceInit,
                           0x409
                           );
```
