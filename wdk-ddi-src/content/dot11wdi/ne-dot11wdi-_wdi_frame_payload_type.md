# _WDI_FRAME_PAYLOAD_TYPE enumeration


## -description

The WDI_FRAME_PAYLOAD_TYPE enumeration defines the frame payload type.

## -enum-fields

### -field WDI_FRAME_MSDU

MAC service data unit (MSDU).

### -field WDI_FRAME_MSDU_FRAGMENT
