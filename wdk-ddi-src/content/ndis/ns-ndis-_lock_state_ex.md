# _LOCK_STATE_EX structure


## -description


The <b>LOCK_STATE_EX</b> structure tracks the state of a read/write lock. This structure is opaque to NDIS
   drivers.


## -struct-fields

### -field OldIrql

Reserved.

### -field LockState

Reserved.

### -field Flags

Reserved.
