# NdisGetPhysicalAddressLow macro


## -description

<b>NdisGetPhysicalAddressLow</b> returns the low-order part of a given physical address.

## -parameters

### -param _PhysicalAddress 

[in]
Specifies a physical address of an OS-dependent size.
