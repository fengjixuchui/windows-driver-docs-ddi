# _NDIS_PD_CLOSE_PROVIDER_PARAMETERS structure


## -description

This structure represents the parameters that are used when calling the <a href="https://docs.microsoft.com/windows-hardware/drivers/network/oid-pd-close-provider">OID_PD_CLOSE_PROVIDER</a> OID.

## -struct-fields

### -field Header

The <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/ntddndis/ns-ntddndis-_ndis_object_header">NDIS_OBJECT_HEADER</a> structure for the <b>NDIS_PD_CLOSE_PROVIDER_PARAMETERS</b> structure. Set the members of this structure as follows:

<ul>
<li><b>Type</b> = <b>NDIS_OBJECT_TYPE_DEFAULT</b></li>
<li><b>Revision</b> = <b>NDIS_PD_CLOSE_PROVIDER_PARAMETERS_REVISION_1</b></li>
<li><b>Size</b> = <b>NDIS_SIZEOF_PD_CLOSE_PROVIDER_PARAMETERS_REVISION_1</b></li>
</ul>

### -field Flags

This member is reserved and must be set to 0.

### -field ProviderHandle

A provider handle that identifies the PD-capable miniport driver's provider object.

## -remarks

This structure must be aligned on an 8-byte boundary.
