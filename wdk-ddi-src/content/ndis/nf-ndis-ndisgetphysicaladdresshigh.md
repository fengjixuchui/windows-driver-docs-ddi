# NdisGetPhysicalAddressHigh macro


## -description


<b>NdisGetPhysicalAddressHigh</b> returns the high-order part of a given physical address.


## -parameters




### -param _PhysicalAddress [in]

Specifies a physical address of an OS-dependent size.
