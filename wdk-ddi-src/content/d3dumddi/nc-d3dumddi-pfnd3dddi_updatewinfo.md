# PFND3DDDI_UPDATEWINFO callback function

## -description

The <i>UpdateWInfo</i> function updates the w range for w buffering.

## -parameters

### -param hDevice

A handle to the display device (graphics context).

### -param Arg2

*pData* [in]

A pointer to a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/d3dumddi/ns-d3dumddi-_d3dddiarg_winfo">D3DDDIARG_WINFO</a> structure that describes the w range for w buffering.

## -returns

<i>UpdateWInfo</i> returns S_OK or an appropriate error result if the w range is not successfully updated.
