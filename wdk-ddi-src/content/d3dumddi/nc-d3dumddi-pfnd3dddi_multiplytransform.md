# PFND3DDDI_MULTIPLYTRANSFORM callback function


## -description

The <i>MultiplyTransform</i> function modifies the current transform.

## -parameters

### -param hDevice

A handle to the display device (graphics context).

### -param Arg2

*pData* [in]

A pointer to a <a href="/windows-hardware/drivers/ddi/d3dumddi/ns-d3dumddi-_d3dddiarg_multiplytransform">D3DDDIARG_MULTIPLYTRANSFORM</a> structure that describes how to modify the current transform.

## -returns

<i>MultiplyTransform</i> returns S_OK or an appropriate error result if the transform is not successfully modified.
