# PFND3DDDI_STATESET callback function


## -description

The <i>StateSet</i> function sets a state block.

## -parameters

### -param hDevice

A handle to the display device (graphics context).

### -param Arg2

*pData* [in]

A pointer to a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/d3dumddi/ns-d3dumddi-_d3dddiarg_stateset">D3DDDIARG_STATESET</a> structure that describes how to set the state block.

## -returns

<i>StateSet</i> returns S_OK or an appropriate error result if the state block is not successfully set.
