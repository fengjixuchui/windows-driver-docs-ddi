# PFND3DDDI_SETTRANSFORM callback function

## -description

The <i>SetTransform</i> function sets up a transform.

## -parameters

### -param hDevice

A handle to the display device (graphics context).

### -param Arg2

*pData* [in]

A pointer to a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/d3dumddi/ns-d3dumddi-_d3dddiarg_settransform">D3DDDIARG_SETTRANSFORM</a> structure that describes how to set up the transform.

## -returns

<i>SetTransform</i> returns S_OK or an appropriate error result if the transform is not successfully set up.
