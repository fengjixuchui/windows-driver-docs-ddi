# _PEP_PPM_CONTEXT_QUERY_PARKING_PAGE structure


## -description


The <b>PEP_PPM_CONTEXT_QUERY_PARKING_PAGE</b> structure describes the parking page for a processor.


## -struct-fields




### -field PhysicalPageAddress

The physical memory address of the parking page.


### -field VirtualPageAddress

The virtual memory address of the parking page.


## -remarks



The output buffer for a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/pepfx/ns-pepfx-_pep_ppm_context_query_parking_page">PEP_PPM_POWER_CONTROL_QUERY_PARKING_PAGE</a> power control request is a <b>PEP_PPM_CONTEXT_QUERY_PARKING_PAGE</b> structure.
