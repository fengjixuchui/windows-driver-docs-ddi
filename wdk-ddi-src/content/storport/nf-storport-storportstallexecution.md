# StorPortStallExecution function


## -description

The <b>StorPortStallExecution</b> routine stalls the miniport driver.

## -parameters

### -param Delay 

[in]
Specifies the delay interval, in microseconds. The given value must be less than a full millisecond.

## -returns

None
