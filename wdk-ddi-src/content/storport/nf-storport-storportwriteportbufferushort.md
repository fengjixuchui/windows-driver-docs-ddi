# StorPortWritePortBufferUshort macro


## -description


The <b>StorPortWritePortBufferUshort</b> routine writes a value to a specified register address. 


## -parameters




### -param h

<p>Pointer to the hardware device extension.</p>


### -param p

<p>Contains the address of the port to be written to. </p>


### -param b

<p>Pointer to the buffer containing the data to be written. </p>


### -param c

<p>Contains the number of data items of size <b>sizeof</b>(USHORT) to be written. </p>






## -remarks



For more information, see <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/srb/nf-srb-scsiportwriteportbufferushort">ScsiPortWritePortBufferUshort</a>. For a non-buffered equivalent of this routine, see <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/storport/nf-storport-storportwriteportushort">StorPortWritePortUshort</a>.
