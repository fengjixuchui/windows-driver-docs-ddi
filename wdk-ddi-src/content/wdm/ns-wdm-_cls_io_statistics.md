# _CLS_IO_STATISTICS structure


## -description


The <b>CLFS_IO_STATISTICS</b> structure holds I/O statistics data for a Common Log File System (CLFS) log. 


## -struct-fields




### -field hdrIoStats

A <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/ns-wdm-_cls_io_statistics_header">CLFS_IO_STATISTICS_HEADER</a> structure that holds header information for the set of statistics.


### -field cFlush

The number of data flushes.


### -field cbFlush

The number of bytes of data flushed.


### -field cMetaFlush

The number of metadata flushes.


### -field cbMetaFlush

The number of bytes of metadata flushed.
