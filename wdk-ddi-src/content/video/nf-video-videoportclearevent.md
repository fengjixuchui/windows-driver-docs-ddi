# VideoPortClearEvent function


## -description

The <b>VideoPortClearEvent</b> function sets a given event object to the nonsignaled state.

## -parameters

### -param HwDeviceExtension 

[in]
Pointer to the miniport driver's device extension.

### -param pEvent 

[in]
Pointer to the event object.

## -returns

None

## -remarks

To set the state of an event object to the signaled state, use <a href="/windows-hardware/drivers/ddi/video/nf-video-videoportsetevent">VideoPortSetEvent</a>.
