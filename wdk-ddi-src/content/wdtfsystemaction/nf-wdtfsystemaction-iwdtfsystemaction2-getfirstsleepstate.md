# IWDTFSystemAction2::GetFirstSleepState


## -description

Returns the first supported sleep state.

## -parameters

### -param pnSleepState 

[out, retval]
The sleep state.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
