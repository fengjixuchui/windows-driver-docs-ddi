# D3D11_1DDI_VIDEO_COLOR_RGBA structure


## -description


Specifies an RGB color value.


## -struct-fields




### -field R

The red value.


### -field G

The green value.


### -field B

The blue value.


### -field A

The alpha value. Values range from 0 (transparent) to 1 (opaque).
