# PFND3D11DDI_PERFORM_AMORTIZED_PROCESSING_CB callback function


## -description


The <b>pfnPerformAmortizedProcessingCb</b> function performs amortized processing. 


## -parameters




### -param Arg1

*hRuntimeDevice* [in]

A handle to a context for the core Microsoft Direct3D runtime. This handle is supplied to the driver in a call to the driver's <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/d3d10umddi/nc-d3d10umddi-pfnd3d10ddi_createdevice">CreateDevice(D3D10)</a> function. 


## -returns



None




## -remarks



For more information about amortized processing, see <a href="https://docs.microsoft.com/windows-hardware/drivers/display/changes-from-direct3d-10">Changes from Direct3D 10</a>.
