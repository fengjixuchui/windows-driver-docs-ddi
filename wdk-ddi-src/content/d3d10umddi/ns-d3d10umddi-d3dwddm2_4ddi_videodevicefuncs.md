# D3DWDDM2_4DDI_VIDEODEVICEFUNCS structure

## -description

Specifies the video function table for the Microsoft Direct3D driver device object. Used only by Windows Display Driver Model (WDDM) 2.4 and later drivers.

>**Note** The driver must support WDDM 2.4 version of the user mode DDI. When the DDI is supported, the runtime will call PFND3D10DDI_RETRIEVESUBOBJECT for D3DWDDM2_1DDI_VIDEO_FUNCTIONS to retrieve the new video device functions.
