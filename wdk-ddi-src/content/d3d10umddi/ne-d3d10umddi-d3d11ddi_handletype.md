# D3D11DDI_HANDLETYPE enumeration


## -description


Contains values that identify handle types.

A resource handle. 


### -field D3D10DDI_HT_SHADERRESOURCEVIEW

### -field D3D10DDI_HT_QUERY

A query handle.


### -field D3D11DDI_HT_COMMANDLIST

### -field D3D11DDI_HT_UNORDEREDACCESSVIEW
