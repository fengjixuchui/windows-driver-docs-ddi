# _CREATE_DISK_GPT structure


## -description


The CREATE_DISK_GPT structure is used with the IOCTL <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/ntdddisk/ni-ntdddisk-ioctl_disk_create_disk">IOCTL_DISK_CREATE_DISK</a> to initialize a disk with an empty GPT partition table.


## -struct-fields




### -field DiskId

Specifies the GUID that uniquely identifies the disk.


### -field MaxPartitionCount

Specifies the maximum number of partitions allowed on the disk.
