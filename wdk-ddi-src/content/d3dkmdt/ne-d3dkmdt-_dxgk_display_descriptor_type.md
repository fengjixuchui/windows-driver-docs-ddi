# _DXGK_DISPLAY_DESCRIPTOR_TYPE enumeration


## -description


Enum used to express the display descriptor type.


## -enum-fields




### -field DXGK_DDT_INVALID

Invalid type.


### -field DXGK_DDT_EDID

A VESA EDID descriptor.

### -field DXGK_DDT_DISPLAYID 

Display id type. Starting in WDDM 2.5.

### -field BYTE

Indicates byte.
