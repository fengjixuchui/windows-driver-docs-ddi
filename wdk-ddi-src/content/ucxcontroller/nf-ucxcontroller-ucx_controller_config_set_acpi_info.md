# UCX_CONTROLLER_CONFIG_SET_ACPI_INFO function


## -description

Initializes a <a href="/windows-hardware/drivers/ddi/ucxcontroller/ns-ucxcontroller-_ucx_controller_config">UCX_CONTROLLER_CONFIG</a> structure with the specified values for the controller with ACPI as the parent.

## -parameters

### -param Config

A pointer to a <a href="/windows-hardware/drivers/ddi/ucxcontroller/ns-ucxcontroller-_ucx_controller_config">UCX_CONTROLLER_CONFIG</a> structure to initialize.

### -param VendorId

A string that contains the vendor identifier for the device.

### -param DeviceId

A string that specifies the device identifier assigned by the manufacturer.

### -param RevisionId

A string that Specifies the revision level of the device described by the <b>DeviceID</b> member.
