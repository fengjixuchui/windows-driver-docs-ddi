# D3D12DDI_SWIZZLE_BIT_ENTRY structure


## -description


Defines a swizzle bit entry.


## -struct-fields




### -field Valid

A valid value.


### -field ChannelIndex

The channel index. Specify zero (0) for X, one (1) for Y, two (2) for Z, and three (3) for SS.


### -field SourceBitIndex

Index of source bit address.
