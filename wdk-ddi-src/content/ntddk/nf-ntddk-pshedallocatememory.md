# PshedAllocateMemory function


## -description


The <b>PshedAllocateMemory</b> function allocates a block of memory from the nonpaged pool.


## -parameters




### -param Size [in]

The size, in bytes, of the block of memory being allocated.


## -returns



<b>PshedAllocateMemory</b> returns a pointer to the allocated block of memory, or NULL if the memory allocation fails.




## -remarks



A PSHED plug-in calls the <b>PshedAllocateMemory</b> function to allocate a block of memory. When the PSHED plug-in is done using the allocated block of memory, it calls the <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddk/nf-ntddk-pshedfreememory">PshedFreeMemory</a> function to free the memory.
