# _FILE_ALIGNMENT_INFORMATION structure


## -description

The <b>FILE_ALIGNMENT_INFORMATION</b> structure is used as an argument to the <a href="/windows-hardware/drivers/ddi/ntifs/nf-ntifs-ntqueryinformationfile">ZwQueryInformationFile</a> routine.

## -struct-fields

### -field AlignmentRequirement

The buffer alignment required by the underlying device. For a list of system-defined values, see <a href="/windows-hardware/drivers/ddi/wdm/ns-wdm-_device_object">DEVICE_OBJECT</a>. The value must be one of the FILE_<i>XXX</i>_ALIGNMENT values defined in Wdm.h. For more information, see <a href="/windows-hardware/drivers/ddi/wdm/ns-wdm-_device_object">DEVICE_OBJECT</a> and <a href="/windows-hardware/drivers/kernel/initializing-a-device-object">Initializing a Device Object</a>.
