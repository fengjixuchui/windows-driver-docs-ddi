# PsIsHostSilo function


## -description

This routine  will check if the supplied <i>Silo</i> is the host silo.

## -parameters

### -param Silo 

[in]
The  silo to evaluate.

## -returns

Returns <b>true</b> if the specified silo is the host silo; otherwise, <b>false</b>.
