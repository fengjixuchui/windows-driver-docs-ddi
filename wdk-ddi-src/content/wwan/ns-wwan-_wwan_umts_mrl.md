# _WWAN_UMTS_MRL structure


## -description


The <b>WWAN_UMTS_MRL</b> structure contains the measured results list (MRL) of neighboring UMTS cells.


## -struct-fields




### -field ElementCount

The count of MRL entries following this member.


### -field UMTSMrl

The array of MRL records, each specified as a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wwan/ns-wwan-_wwan_umts_mrl_info">WWAN_UMTS_MRL_INFO</a> structure.
