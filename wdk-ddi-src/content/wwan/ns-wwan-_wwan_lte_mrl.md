# _WWAN_LTE_MRL structure


## -description


The <b>WWAN_LTE_MRL</b> structure represents the measured results list (MRL) of neighboring LTE cells.


## -struct-fields




### -field ElementCount

The count of MRL entries following this member.


### -field LTEMrl

The array of MRL records, each specified as a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/wwan/ns-wwan-_wwan_lte_mrl_info">WWAN_LTE_MRL_INFO</a> structure.
