# _SetDSMCounters_IN structure


## -description


The SetDSMCounters_IN structure is used to set the timer counters for a particular DSM.


## -struct-fields




### -field DsmContext

A 64-bitfield that provides the DSM context.


### -field DsmCounters

A structure of type DSM_COUNTERS.
