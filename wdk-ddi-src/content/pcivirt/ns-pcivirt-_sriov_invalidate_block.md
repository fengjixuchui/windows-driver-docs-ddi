# _SRIOV_INVALIDATE_BLOCK structure


## -description


Contains the configuration block information. This structure is used in a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/pcivirt/ni-pcivirt-ioctl_sriov_invalidate_block">IOCTL_SRIOV_INVALIDATE_BLOCK</a> request.


## -syntax


```cpp
typedef struct _SRIOV_INVALIDATE_BLOCK {
  USHORT  VfIndex;
  UINT64  BlockMask;
} SRIOV_INVALIDATE_BLOCK, SRIOV_INVALIDATE_BLOCK;
```


## -struct-fields




### -field VfIndex

Zero-based index of the virtual function (VF) from the first VF exposed by this physical function (PF).


### -field BlockMask

a block of configuration data.
