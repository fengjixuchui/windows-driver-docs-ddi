# IDebugSymbols3::SetScopeFromStoredEvent


## -description

The <b>SetScopeFromStoredEvent</b> method sets the current scope to the scope of the stored event.

This method may also return error values.  See <a href="/windows-hardware/drivers/debugger/hresult-values">Return Values</a> for more details.

## -remarks

Currently only user-mode <a href="/windows-hardware/drivers/debugger/user-mode-dump-files#minidumps">Minidumps</a> can contain a stored event.

The new scope is printed to the debugger console.

For more information about scopes, see <a href="/windows-hardware/drivers/debugger/scopes-and-symbol-groups">Scopes and Symbol Groups</a>.
