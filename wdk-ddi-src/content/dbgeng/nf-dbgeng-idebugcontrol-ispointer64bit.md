# IDebugControl::IsPointer64Bit


## -description


The <b>IsPointer64Bit</b> method determines if the effective processor uses 64-bit pointers.

</dl>
</td>
<td width="60%">

</tr>
<tr>
<td width="40%">
<dl>
<dt><b>S_FALSE</b></dt>
</dl>

<td width="60%">
