# _DEBUG_PROCESSOR_IDENTIFICATION_X86 structure


## -description


Identifies an x86 processor. 


## -struct-fields




### -field Family

The family of the processor.


### -field Model

The model of the processor.


### -field Stepping

The stepping value of the processor.


### -field VendorString

A vendor specified string.
