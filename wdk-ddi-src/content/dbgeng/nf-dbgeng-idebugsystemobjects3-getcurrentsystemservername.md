# IDebugSystemObjects3::GetCurrentSystemServerName


## -description

Gets the server name for the current process.

### -param Buffer 

[out]
A pointer to an output buffer.

### -param BufferSize 

[in]
The size of the buffer.

### -param NameSize 

[out, optional]
A pointer to the name size.
