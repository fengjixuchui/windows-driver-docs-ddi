# IDebugControl3::GetCurrentEventIndex


## -description

The <b>GetCurrentEventIndex</b> method returns the index of the current event within the current list of events for the current target, if such a list exists.

## -parameters

### -param Index 

[out]
Receives the index of the current event in the target.  The index will be a number between zero and one less than the number of events returned by <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/dbgeng/nf-dbgeng-idebugcontrol3-getnumberevents">GetNumberEvents</a>.  The index of the first event is zero.

## -returns

This method may also return error values.  See <a href="https://docs.microsoft.com/windows-hardware/drivers/debugger/hresult-values">Return Values</a> for more details.

## -remarks

Targets that do not have fixed sets of events will always return zero to <i>Index</i>.
