# _SetPersistentEntry_IN structure


## -description

The SetPersistentEntry_IN structure is used by a WMI client to deliver the input parameter data of the <a href="/windows-hardware/drivers/storage/setpersistententry">SetPersistentEntry</a> WMI method to the HBA miniport driver.

## -struct-fields

### -field PortWWN

Contains a worldwide name that indicates the port whose persistent bindings will be changed.

### -field Binding

Contains a structure of type <a href="/windows-hardware/drivers/ddi/hbapiwmi/ns-hbapiwmi-_hbafcpbindingentry2">HBAFCPBindingEntry2</a> that indicates the binding to be removed from the indicated port's list of bindings.

## -remarks

The WMI tool suite generates a declaration of the SetPersistentEntry_IN structure in <i>Hbapiwmi.h </i>when it compiles the <a href="/windows-hardware/drivers/storage/msfc-hbafcpinfo-wmi-class">MSFC_HBAFCPInfo WMI Class</a>.
