# _tagKSTOPOLOGY_ENDPOINTID structure


## -description


The <b>KSTOPOLOGY_ENDPOINTID</b> structure specifies the name and the pin ID of a topology endpoint.


## -struct-fields




### -field TopologyName

The name of the topology endpoint.


### -field PinId

The pin ID of the topology endpoint.
