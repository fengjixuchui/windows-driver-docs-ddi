# IWDTFActions2::Add


## -description

Add a single action to the collection.

## -parameters

### -param pAction 

[in]
The action to add to this collection.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
