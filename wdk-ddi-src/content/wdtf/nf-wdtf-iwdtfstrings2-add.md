# IWDTFStrings2::Add


## -description

Adds a single string to the collection.

## -parameters

### -param pString 

[in]
The string to add to this collection.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
