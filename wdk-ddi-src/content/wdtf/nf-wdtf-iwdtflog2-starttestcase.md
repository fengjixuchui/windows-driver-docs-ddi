# IWDTFLOG2::StartTestCase


## -description

Marks the start of a test case.

## -parameters

### -param sTestCaseName 

[in]
The test case name.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
