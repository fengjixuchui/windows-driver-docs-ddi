# _USBC_DATA_ROLE enumeration

## -description
Defines data roles of USB Type-C connected devices.


## -enum-fields

### -field UsbCDataRoleInvalid 

Indicates the data role state is invalid.

### -field UsbCDataRoleDfp 
Operates as a DFP.

### -field UsbCDataRoleUfp 
Operates as a UFP.
