# KsRemoveSpecificIrpFromCancelableQueue function


## -description


The <b>KsRemoveSpecificIrpFromCancelableQueue</b> function removes the specified IRP from the specified queue. This is performed on an IRP that was previously acquired using <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ks/nf-ks-ksremoveirpfromcancelablequeue">KsRemoveIrpFromCancelableQueue</a>, but that was not actually removed from the queue. 


## -parameters




### -param Irp [in]

Points to I/O request packet.


## -returns



None
