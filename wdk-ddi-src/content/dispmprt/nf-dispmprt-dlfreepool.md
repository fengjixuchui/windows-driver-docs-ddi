# DlFreePool function


## -description

Deallocates a block of pool memory previously allocated by the [DlAllocatePool](nf-dispmprt-dlallocatepool.md) function.

## -parameters

### -param DeviceHandle

A handle that represents a display adapter.

### -param Ptr

Pointer to a memory pool.
