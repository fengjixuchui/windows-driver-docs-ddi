# DlUnlockBuffer function


## -description

Unlocks physical pages described by the specified memory descriptor list.

## -parameters

### -param DeviceHandle

A handle that represents a display adapter.

### -param Mdl

The memory descriptor list.


## -returns

This function returns VOID.
