# SENSOR_CONFIG_INIT function


## -description


This function initializes a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/sensorscx/ns-sensorscx-_sensor_config">SENSOR_CONFIG</a> structure.


## -parameters




### -param pConfig [out]
