# FsRtlCompleteRequest macro


## -description


The <b>FsRtlCompleteRequest</b> macro completes an IRP with the specified status.


## -parameters




### -param IRP

<p>Pointer to the IRP to be completed.</p>


### -param STATUS

<p>Status value to be returned for the IRP.</p>
