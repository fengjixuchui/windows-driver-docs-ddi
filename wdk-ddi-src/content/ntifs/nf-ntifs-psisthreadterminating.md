# PsIsThreadTerminating function


## -description


The <b>PsIsThreadTerminating</b> routine checks whether a thread is terminating.


## -parameters




### -param Thread [in]

A pointer to the thread to be checked for termination.


## -returns



The <b>PsIsThreadTerminating</b> routine returns <b>TRUE</b> if the thread is terminating, otherwise <b>FALSE</b>.




## -remarks



For more information about using system threads and managing synchronization within a nonarbitrary thread context, see <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/index">Driver Threads, Dispatcher Objects, and Resources</a>.
