# IoIsSystemThread function


## -description

The <b>IoIsSystemThread</b> routine checks whether a given thread is a system thread.

## -parameters

### -param Thread 

[in]
Pointer to the thread to be checked.

## -returns

<b>IoIsSystemThread</b> returns <b>TRUE</b> if the specified thread is a system thread, otherwise <b>FALSE</b>.

## -remarks

For more information about using system threads and managing synchronization within a nonarbitrary thread context, see <a href="/windows-hardware/drivers/ddi/index">Driver Threads, Dispatcher Objects, and Resources</a>.
