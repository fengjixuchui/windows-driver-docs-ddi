# D3DKMTPresentMultiPlaneOverlay2 function


## -description

Pointer to a [D3DKMT_PRESENT_MULTIPLANE_OVERLAY2](ns-d3dkmthk-_d3dkmt_present_multiplane_overlay2.md) structure.
