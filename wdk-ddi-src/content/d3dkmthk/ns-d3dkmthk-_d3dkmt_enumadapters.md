# _D3DKMT_ENUMADAPTERS structure


## -description


Supplies information for  enumerating all graphics adapters on the system.


## -struct-fields




### -field NumAdapters

[in] The number of graphics adapters.


### -field Adapters

An array of <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_adapterinfo">D3DKMT_ADAPTERINFO</a> structures that supply configuration information for each adapter. The maximum number of adapters that can be enumerated is 16.
