# D3DKMTGetDeviceState function

## -description

The <b>D3DKMTGetDeviceState</b> function retrieves the state of a device.

## -parameters

### -param Arg1

*pData* [in, out]

[in/out] A pointer to a <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_getdevicestate">D3DKMT_GETDEVICESTATE</a> structure that describes parameters for retrieving the state of a device.

## -returns

<b>D3DKMTGetDeviceState</b> returns one of the following values:

|Return code|Description|
|--- |--- |
|STATUS_SUCCESS|The state of the device was successfully retrieved.|
|STATUS_DEVICE_REMOVED|The graphics adapter was stopped or the display device was reset.|
|STATUS_INVALID_PARAMETER|Parameters were validated and determined to be incorrect.|

This function might also return other <b>NTSTATUS</b> values.
