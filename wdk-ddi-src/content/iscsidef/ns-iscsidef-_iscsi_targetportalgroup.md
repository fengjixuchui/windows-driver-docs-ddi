# _ISCSI_TargetPortalGroup structure


## -description


The ISCSI_TargetPortalGroup structure provides a definition of a target portal group. 


## -struct-fields




### -field PortalCount

The number of portals in the portal group. 


### -field Portals

A variable-length array of <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsidef/ns-iscsidef-_iscsi_targetportal">ISCSI_TargetPortal</a> structures, which describe portals in the target portal group. The number of elements in the array is specified by the PortalCount field.
