# IWDTFPNPActions2::RemoveDevice


## -description

Removes the device.

## -parameters

### -param pbSuccess 

[out, retval]
True if the operation succeeds; otherwise, false.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.

## -remarks

Under the covers, this will call DIF_REMOVE.
