# _COMPOSITE_DEVICE_CAPABILITIES structure


## -description


The <b>COMPOSITE_DEVICE_CAPABILITIES</b> structure specifies the capabilities of the  driver of a USB multi-function device (composite driver). To initialize the structure, use the <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/usbdlib/nf-usbdlib-composite_device_capabilities_init">COMPOSITE_DEVICE_CAPABILITIES_INIT</a> macro.


## -struct-fields




### -field CapabilityFunctionSuspend

If set to 1, indicates that the composite driver supports the USB function suspend feature. 


### -field Reserved

Reserved.
