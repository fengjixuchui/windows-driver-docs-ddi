# _SYSENV_VARIABLE structure


## -description


Stores the name a system environment variable using
    SysEnv device. This structure is used in the <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/ntddsysenv/ni-ntddsysenv-ioctl_sysenv_get_variable">IOCTL_SYSENV_GET_VARIABLE</a> request.


## -struct-fields




### -field VendorGuid

The vendor GUID.


### -field VariableNameLength

The length of the string pointed to by  <b>VariableName</b>, which contains the name of the variable.


### -field VariableName

A string that  contains the name of the variable.
