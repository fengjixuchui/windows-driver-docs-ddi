# _SECURE_ELEMENT_EVENT_SUBSCRIPTION_INFO structure


## -description


The SECURE_ELEMENT_EVENT_SUBSCRIPTION_INFO structure is an input parameter to <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/nfcsedev/ni-nfcsedev-ioctl_nfcse_subscribe_for_event">IOCTL_NFCSE_SUBSCRIBE_FOR_EVENT</a>.


## -struct-fields




### -field guidSecureElementId


### -field eEventType

Secure element event type. A service can subscribe and receive notification when an external reader arrival, external reader departure, transaction, HCE activated, or HCE deactivated event is triggered.
